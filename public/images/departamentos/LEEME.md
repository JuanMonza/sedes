# Imágenes de banderas / insignias departamentales

Coloca aquí las imágenes de las banderas o escudos de cada departamento.
El componente `DepartamentoFlipCard` las carga automáticamente usando el slug del departamento.

## Archivos esperados (formato PNG recomendado)

| Departamento     | Nombre de archivo        |
|------------------|--------------------------|
| Antioquia        | `antioquia.png`          |
| Arauca           | `arauca.png`             |
| Atlántico        | `atlantico.png`          |
| Bogotá D.C       | `bogota-dc.png`          |
| Cauca            | `cauca.png`              |
| Cundinamarca     | `cundinamarca.png`       |
| Huila            | `huila.png`              |
| Meta             | `meta.png`               |
| Risaralda        | `risaralda.png`          |
| Santander        | `santander.png`          |
| Tolima           | `tolima.png`             |
| Valle del Cauca  | `valle-del-cauca.png`    |

## Dónde conseguirlas

- Wikipedia Commons: https://commons.wikimedia.org/wiki/Flags_of_Colombia
- Tamaño recomendado: 400 × 250 px mínimo, fondo transparente (PNG)
- SVG también funciona: cambia la extensión en `DepartamentoFlipCard.tsx` si usas SVG

## Nota

Si un archivo no existe, el card sigue mostrándose correctamente con el color
de acento del departamento como fondo. No hay imagen rota visible.
